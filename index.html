Common CSS pseudo-classes

Pseudo-class Description
:hover The element is hovered over.
:active The element is active.
:focus The element is focused.
:visited The element is visited.
:link The element is linked.
:first-child The element is the first child of its parent.
:last-child The element is the last child of its parent.
:first-of-type The element is the first of its type in the list of children of its parent.
:last-of-type The element is the last of its type in the list of children of its parent.
:only-child The element is the only child of its parent.
:only-of-type The element is the only of its type in the list of children of its parent.
:nth-child The element is the nth child of its parent.
